## What is this?

A collection of tools to help Swedish voters navigate political discourse with wisdom, empathy, and critical thinking.

## Features

- **Article Analyzer**: AI-powered analysis of political journalism
- **Manifesto Comparison**: Side-by-side party program comparisons
- **Educational Content**: Stories and frameworks for thoughtful political engagement
- **Bilingual**: Full Swedish and English support

## Technology

- Pure client-side React (no backend required)
- Tailwind CSS for styling
- Lucide icons
- 100% free to host and run

## Philosophy

- **Politically neutral**: No stance for or against any party
- **Open source**: Transparent code and methodology
- **Wisdom over virality**: Optimized for learning, not engagement
- **Empathy-based**: Assumes good faith from all voters
